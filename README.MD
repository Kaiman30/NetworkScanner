# Запускать от имени администратора

После запуска создает файл `network_checker_report.html` и открывает его в браузере для просмотра отчета.
В случае, если браузер не открылся - следуйте инструкции ниже:

1. Нажимаете сочетание клавиш `Win + R`
2. Вставляете в открывшееся окно строку - `"C:\Users\%username%\AppData\Local\Temp\network_checker_report.html"`
3. Нажимаете `Enter`

## Важно

> Наличие обнаруженных настроек не гарантирует, что нужно забанить подозреваемого за них. Следует внимательно изучить, что именно настроено и какое значение выставлено для того или иного пункта
